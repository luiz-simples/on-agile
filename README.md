OnAgile
========

Install & Run
-----------
  1. Install NodeJS: http://nodejs.org/
  2. run this code block in your terminal

```
git clone [git@github.com:luiz-simples/on-agile.git](git@github.com:luiz-simples/on-agile.git)
cd on-agile
npm install
gulp tests
```
